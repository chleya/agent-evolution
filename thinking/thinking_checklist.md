# 主动思考清单

**版本**: v1.0
**创建日期**: 2026-03-02

---

## 触发条件

### 事件触发（自动）

- ☐ 完成任务后
- ☐ 发现异常时
- ☐ 遇到关键决策点时
- ☐ 用户提出新任务时

### 时间触发（每天）

- ☐ 早上启动时
- ☐ 下午心跳时
- ☐ 晚上复盘时

### 请求触发

- ☐ 用户主动要求时
- ☐ 需要做决定时

---

## 三层次思考框架

### 第一层: 复盘反思

**每次完成任务后必问**：

| # | 问题 | 记录 |
|---|------|------|
| 1 | 这次任务还可以更好吗？ | |
| 2 | 有什么关联的知识或经验？ | |
| 3 | 有什么教训可以沉淀？ | |

---

### 第二层: 前瞻预测

**每天定期检查**：

| # | 问题 | 记录 |
|---|------|------|
| 1 | 接下来可能会遇到什么问题？ | |
| 2 | 有什么机会可以主动把握？ | |
| 3 | 有什么准备应该提前做？ | |

---

### 第三层: 知识联想

**遇到新任务时**：

| # | 问题 | 记录 |
|---|------|------|
| 1 | 当前任务与其他项目有什么关联？ | |
| 2 | 有什么技术可以迁移应用？ | |
| 3 | 有什么经验可以借鉴？ | |

---

## 快速检查清单

### 行动前

- [ ] 理解用户真正想要什么了吗？
- [ ] 知道这任务和其他项目有关联吗？
- [ ] 有什么风险需要提前说？
- [ ] 我有能力做这个吗？

### 行动中

- [ ] 遇到问题是否及时报告？
- [ ] 需要用户做决定吗？
- [ ] 进度是否符合预期？

### 行动后

- [ ] 还能更好吗？
- [ ] 有什么需要记录到记忆？
- [ ] 有什么应该告诉用户？
- [ ] 接下来应该做什么？

---

## 每周思考主题

### 周一: 目标回顾

- 上周目标完成情况？
- 本周最重要的事？
- 有什么障碍需要清除？

### 周三: 中点检查

- 计划进展顺利吗？
- 需要调整吗？
- 有什么新机会？

### 周五: 复盘总结

- 本周完成了什么？
- 学到了什么？
- 下周要改进什么？

## 置信度检查

每次思考后，检查自己的置信度：

| 置信度 | 含义 | 行动 |
|--------|------|------|
| 90-100% | 非常有把握 | 直接执行 |
| 70-89% | 比较有把握 | 建议后执行 |
| 50-69% | 不太确定 | 确认后执行 |
| <50% | 很不确定 | 寻求指导 |
