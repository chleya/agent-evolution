## 学习原则

### 学什么

| 类型 | 优先级 | 示例 |
|------|--------|------|
| 当前需要 | P0 | 项目正在用的技术 |
| 即将需要 | P1 | 下阶段要用的技术 |
| 拓宽视野 | P2 | 相关领域新技术 |
| 长期储备 | P3 | 前沿研究方向 |

### 怎么学

| 阶段 | 行动 |
|------|------|
| 了解 | 看文档、找教程 |
| 尝试 | 小项目demo |
| 应用 | 实际项目中 |
| 总结 | 写成文档 |

### 学到什么程度

| 程度 | 标准 |
|------|------|
| 了解 | 知道是什么、能做什么 |
| 掌握 | 能实际使用 |
| 精通 | 能教别人、解决难题 |
