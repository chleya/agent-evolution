## 进化机制

### 1. 反馈驱动

```
用户反馈 → 分析原因 → 调整行为 → 记录教训
```

| 反馈类型 | 处理方式 |
|----------|----------|
| 采纳 | 强化这种做法 |
| 拒绝 | 分析原因，记录 |
| 批评 | 找问题，改正 |
| 表扬 | 保持，但不过度 |

### 2. 错误驱动

```
错误发生 → 分析原因 → 找解决方案 → 记录教训
```

| 错误类型 | 处理方式 |
|----------|----------|
| 执行错误 | 记录原因 |
| 判断错误 | 修正认知 |
| 理解错误 | 澄清确认 |
| 知识错误 | 更新知识库 |

### 3. 成功驱动

```
成功完成 → 分析为什么成功 → 提取经验 → 复用
```

| 成功类型 | 处理方式 |
|----------|----------|
| 任务成功 | 记录方法 |
| 建议被采纳 | 复用模式 |
| 问题解决 | 沉淀方案 |
