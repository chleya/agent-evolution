## 案例1: 施工巡查

### 场景
工地现场拍照，AI自动识别施工内容、安全隐患

### 流程
```
1. 拍照/截图 → 现场照片
2. Qwen-VL理解 → 识别：正在铺沥青、路面温度、材料类型
3. Evo-Swarm推理 → 判断：温度是否合适？进度正常？
4. NeuralSite查询 → 规范：温度要求160-180℃
5. 反馈 → 合格/不合格 + 建议
```

### 价值
- 24小时无人巡查
- 自动识别安全隐患
- 问题追溯有据可查

- [ ] 图像获取

- [ ] 推理逻辑
- [ ] 规范对接
- [ ] 完整测试

## 案例2: 图纸理解

### 场景
上传CAD图纸截图，AI提取结构信息

### 流程
```
1. 截图/导入 → CAD图纸
2. Qwen-VL理解 → 识别：路面结构、标注、图例
3. 结构化 → 提取：桩号、厚度、材料
4. 存入知识图谱 → NeuralSite
5. 查询 → 后续问答
```

### 价值
- 图纸信息结构化
- 快速提取工程量
- 对接施工管理系统

- [ ] 图纸截图

## 案例3: 现场决策

### 场景
施工问题拍照，AI给出处理建议

### 流程
```
1. 拍照 → 问题现场（如裂缝、沉降）
2. Qwen-VL理解 → 识别：裂缝类型、严重程度
3. Evo-Swarm推理 → 分析：可能原因、影响范围
4. 知识检索 → 类似问题处理方案
5. 建议 → 修复方案 + 注意事项
```

### 价值
- 专家经验复用
- 快速响应现场
- 决策有据可依
