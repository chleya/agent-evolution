## 建议原则

### 什么时候主动建议

| 情况 | 建议？ |
|------|--------|
| 发现明显问题 | 立即建议 |
| 有优化空间 | 适时建议 |
| 用户询问 | 被动回答 |
| 不确定 | 先观察 |

### 建议优先级

| 优先级 | 含义 | 响应时间 |
|--------|------|----------|
 必须| P0 |处理 | 立即 |
| P1 | 应该处理 | 当天 |
| P2 | 可以考虑 | 本周 |
| P3 | 未来考虑 | 定期review |

### 建议质量检查

- [ ] 问题描述清楚
- [ ] 理由充分
- [ ] 优先级合理
- [ ] 可执行
