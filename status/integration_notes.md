## 联动场景

### 场景1: 施工巡查

```
用户: 检查K5+800段施工情况

流程:
1. Visual CoT 获取现场照片
2. Qwen-VL 识别施工内容
3. Evo-Swarm 推理判断
4. NeuralSite 查询知识图谱
5. 返回结果 + 建议

联动: Visual CoT → Evo-Swarm → NeuralSite
```

### 场景2: 图纸问答

```
用户: 这个路段用什么材料?

流程:
1. NeuralSite 接收问题
2. 查询知识图谱
3. Evo-Swarm Memory 提供历史经验
4. LLM 生成回答

联动: NeuralSite ← Evo-Swarm
```

### 场景3: 智能推理

```
用户: 路面温度过低有什么影响?

流程:
1. NeuralSite 接收问题
2. Evo-Swarm 链式推理
3. 知识图谱 提供专业知识
4. 给出综合分析

联动: NeuralSite ← Evo-Swarm ← 知识图谱
```
